# ch 1 컴퓨터 비전과 영상의 이해

### 컴퓨터 비전기술이 적용된 제품을 찾아 제품의 기능, 동작원리 등을 조사하여 제출하시오.

1. 공간 컴퓨터 : 애플 비전 프로(Apple Vision Pro)

기능 : 공간 컴퓨팅을 통해 눈, 손, 목소리를 이용한 인터페이스를 제공하고, 가상 화면을 실제 공간에 고정하여 고해상도 3D 영상 시청 및 협업 도구로 사용이 가능하다.

동작 원리 : 

- 12개의 카메라와 5개의 센서(LiDAR, TrueDepth) 등의 센서 퓨전을 통해 실시간으로 주변 환경을 스캔한다.
- Visual SLAM을 사용하여 카메라 영상으로 사용자의 위치 공간 구조를 파악하여 가상 물체를 실제 영상에 고정한다.
1. 휴머노이드 로봇 : 테슬라 옵티머스(Tesla Optimus)

기능 : 테슬라의 자율주행(FSD : Full Self Driving) 기술을 로봇으로 옮겨 물건 운반, 정밀 부품 조립, 가사 노동 보조 등의 작업 수행

동작 원리 : 

- LiDAR 없이 카메라 데이터만으로 주변을 인식하는 Pure Vision을 사용한다.
- 가메라의 2D 영상을 3D 공간의 데이터로 변환하여 로봇의 위치와 공간을 실시간으로 매핑
- 시각 정보 입력 시 딥러닝 모델이 직접 제어 신호를 생성하는 End-to-End 사용
1. 지능형 홈 로봇 : LG AI 홈 허브 (집사 로봇)

기능 : 단순히 청소기를 넘어 집안의 가전과 환경을 관리하는 로봇

동작 원리 : 

- 딥러닝 모델을 사용하여 객체 인식 및 분류
- 가메라 영상을 픽셀 단위로 분석하는 Sematic Segmentation을 사용하여 최적의 이동 경로 설계
- 카메라로 사용자 표정을 읽는 비전 기술과 음성 인식 기술을 결합한 멀티모달 AI를 사용

### 영상처리와 컴퓨터 비전의 차이점을 자세히 설명하시오.

영상 처리(Image Processing) : 이미지를 입력받아 새로운 이미지를 만들어내는 기술로 주로 이미지 품질을 높이기 위한 노이즈 제거, 필터링, 색상 변환, 압축 등의 기능이 있다.

컴퓨터 비전(Computer Vision) : 이미지나 비디오를 입력받아 그 안의 의미나 정보를 추출하는 기술로 컴퓨터가 인간처럼 시각 세계를 이해하도록 만드는 것으로 객체 검출, 이미지 분류, 안면 인식, 동작 분석 등 기능을 있다.

### CMOS 센서의 원리를 자세히 조사하시오.

CMOS(Complementary Metal-Oxide-Semiconductor)  센서 : 인간의 망막과 같이 외부의 빛을 컴퓨터가 이해할 수 있는 디지털 데이터로 변환하는 하드웨어

동작 원리 :

1. 광전 효과를 통한 전하 생성 : 렌즈를 통과한 빛이 각 픽셀의 포토다이오드에 도달 시 광전 효과에 의해 전자를 생성
2. 전하를 전압으로 변환 및 증폭 : CMOS는 각 픽셀 내부에 증폭기가 있어 포토다이오드의 전하량을 전압으로 변환하고 이 과정에서 신호를 증폭하여 노이즈를 억제
3. 아날로드-디지털 변환(ADC) : 증폭된 아날로그 전압 신호를 디지털 값으로 변환
4. 디지털 신호 출력 및 전송 : 변환된 디지털 데이터는 ISP(Image Signal Processor)로 전달되어 이미지 파일 형태로 합성

### 영상의 해상도와 이미지 센서의 개수의 관계를 설명하시오.

해상도 = ‘가로 픽셀 수 x 세로 픽셀 수’

이미지 센서 하나에는 수백만 개의 수광 소자가 있어 해상도가 높다는 것은 이미지 센서 안에 더 많은 픽셀이 들어있다는 것을 뜻한다.

픽셀 수가 많을수록 피사체를 세밀하게 기록할 수 있으므로, 고해상도 결과를 얻을 수 있다.

하지만, 단순히 픽셀 수만 늘리면 픽셀 하나의 크기는 작아지고 받아들이는 빛의 광자 수가 줄어들어 저조도 환경에서 노이즈를 발생시키는 문제가 발생한다.

### 100 x 100 그레이스케일 영상 파일의 크기(바이트)를 계산하라.

100 x 100 영상 파일은 10000개의 픽셀을 가지고 있다.

그레이스케일 영상은 하나의 픽셀의 정수 값 저장을 위해 1바이트의 크기를 가지는 unsigned char 자료형을 사용하므로 100 x 100 그레이스케일 영상 파일의 크기는 10000byte(10KB)이다.

### 100 x 100 컬러 영상 파일의 크기(바이트)를 계산하라.

100 x 100 영상 파일은 10000개의 픽셀을 가지고 있다.

컬러 영상은 하나의 픽셀의 정수 값 저장을 위해 1바이트의 크기를 가지는 unsigned char 자료형을  3개 사용하여 Red, Green, Blue 색상 성분을 저장하브로  100 x 100 컬러 영상 파일의 크기는 30000byte(30KB)이다.
